# application
REST API for analyzing information about products

## Assignment description
A web application should be developed that contains the endpoint /text/analyze.
This endpoint accepts a JSON body containing various metadata about products. An example of this body is in the sampleProductsData.json file.
The response should include the following information:
- Most expensive product
- Cheapest product
- Most sold product
- All names of German products
- All names of Chinese products
- Whether the file contains one or more fragile products
